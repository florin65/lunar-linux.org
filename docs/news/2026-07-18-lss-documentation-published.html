LSS documentation published

A new Lunar Scripts System documentation set is now available on the Lunar Linux Website.

The documentation is organized as a family of complementary guides:

• User Guide

• Administrator and Recovery Guide

• Module Author Guide

• Debugging and Inspection Guide

• Plugin and Extension Guide

• Command and State Reference

Together, these pages cover normal module administration, dependency and configuration management, package state, recovery, Moonbase module development, advanced inspection, plugin behavior and the main LSS commands and file locations.

The documentation is available through the LSS documentation hub.

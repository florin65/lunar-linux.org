Lunar Linux is a source-based, rolling Linux distribution for users who want to understand, build, and control their system from the ground up.

Download Lunar Read the docs

Why Lunar Linux?

Source-based

Build the system from source and keep full control over compile-time choices.

Rolling release

Stay current through Lunar’s own package tools and the moonbase module collection.

Transparent

Simple module files, visible patches, and a build process that stays close to the user.

Community driven

Maintained by a small group of users and developers who value traditional Linux principles.

Built for control

Lunar Linux does not try to hide the system from you. It gives you tools to build, update, inspect, and tune your installation.

$ lunar update $ lin moonbase $ lvu search kernel

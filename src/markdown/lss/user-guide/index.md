# LSS User Guide

This guide covers normal LSS administration:

- installing, rebuilding, and removing modules;
- inspecting package and dependency state;
- managing required and optional dependencies;
- changing build configuration;
- using held, exiled, and enforced states;
- updating Lunar Linux safely.

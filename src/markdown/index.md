# Why Lunar Linux?

- Build from source
- Target-machine optimization
- Optional dependency control
- Minimal downstream patching
- Close-to-upstream software
- Rolling release model
- KISS-oriented design
- Shell-script driven modules and package management system

Designed for administrators who want to stay close to the hardware and Linux systems they manage.
